# MSc Data Science
Group of scripts used for Masters degree in Data Science

UK Mobile phone companies typically create coverage maps based on 50x50m 'tiles'. 
To determine whether there is sufficient signal strength for home fixed broadband it is preferable to work to finer resolution. 
The dissertation and project researched whether simple interpolation algorithms could improve classification accuracy
Method was to compare both non-interpolated predictions and interpolated ones against empirical data commercially obtained from Ookla speed tests.
Note the data is NOT in the public domain. Security credentials have been removed from the download script. Sorry - not my data to share.
